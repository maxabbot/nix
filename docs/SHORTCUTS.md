# Hyprland Shortcuts

Gruvbox Material Dark · home-desktop

## Window Manager

| Key | Action |
|---|---|
| **Apps** | |
| Super + Return | Terminal (kitty) |
| Super + E | Files (nautilus) |
| Super + B | Browser (zen) |
| Super + D | Launcher (fuzzel) |
| **Windows** | |
| Super + Q | Close window |
| Super + F | Fullscreen |
| Super + Shift + F | Fake fullscreen |
| Super + Space | Toggle float |
| Super + P | Pseudo tile |
| Super + J | Toggle split |
| **Focus** | |
| Super + ← ↑ ↓ → | Focus direction |
| Super + Shift + ← ↑ ↓ → | Move window |
| Super + Ctrl + ← ↑ ↓ → | Resize window |
| **Workspaces** | |
| Super + 1-9 | Switch workspace |
| Super + Shift + 1-9 | Move window to workspace |
| Super + Scroll | Next / prev workspace |
| Super + S | Toggle scratchpad |
| Super + Shift + S | Send to scratchpad |
| **System** | |
| Super + L | Lock screen |
| Super + Shift + L | Suspend |
| Super + Shift + E | Logout menu |
| Super + N | Toggle notifications |
| Super + Shift + V | Clipboard history |
| Super + Shift + G | Toggle gaming mode |
| **Screenshots** | |
| Print | Region → clipboard |
| Shift + Print | Screen → clipboard |
| Super + Print | Region → file |

## Zsh / CLI

| Key | Action |
|---|---|
| Ctrl + R | History search (fzf) |
| Ctrl + T | File search (fzf) |
| Alt + C | cd into dir (fzf) |
| `z <query>` | Jump to frecent dir |
| `nixup` | Rebuild NixOS (nh) |
| `nix shell nixpkgs#X` | Ephemeral package |
| `nix develop` | Enter dev shell |
